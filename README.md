# SCCIPC智能车实验室订单与账单管理系统

## 📋 项目简介

这是一个专为SCCIPC智能车实验室设计的财务管理系统，支持收支记录、订单管理、财务分析等功能。

## ✨ 主要功能

### 💰 财务管理
- **收支记录** - 支持收入、支出、订单三种类型记录
- **分类管理** - 实验器材、维护费用、项目资金等分类
- **财务统计** - 实时显示总余额、收入、支出统计
- **月度分析** - 当月收支情况和净收入计算

### 📊 数据操作
- **筛选搜索** - 按类型、分类、时间范围、关键词筛选
- **排序功能** - 按日期、金额、类型等字段排序
- **批量操作** - 支持全选、批量删除功能
- **分页显示** - 每页10条记录，支持翻页

### 📁 导入导出
- **支付宝导入** - 支持CSV格式交易明细导入
- **微信导入** - 支持XLSX格式账单流水导入
- **数据导出** - 支持CSV和JSON格式导出
- **数据备份** - 本地存储自动保存

### 🎨 用户体验
- **响应式设计** - 完美适配桌面和移动设备
- **现代化UI** - 渐变色彩、圆角设计、阴影效果
- **PWA支持** - 可安装为桌面应用
- **离线使用** - 支持离线数据管理

**网站地址**: https://onemott.github.io/personal-finance-manager/

## 💻 本地部署

1. 克隆仓库
```bash
git clone https://github.com/onemott/personal-finance-manager.git
cd personal-finance-manager
```

2. 启动本地服务器
```bash
# 使用Python
python -m http.server 8080

# 或使用Node.js
npx serve .
```

3. 访问 http://localhost:8080

## 📱 技术栈

- **前端**: HTML5 + CSS3 + 原生JavaScript
- **存储**: LocalStorage API
- **图标**: Font Awesome 6.0
- **文件解析**: XLSX.js (Excel文件支持)
- **PWA**: Web App Manifest

## 🔧 文件结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script_final.js     # 主要功能脚本
├── manifest.json       # PWA配置
└── README.md          # 项目说明
```

### 智能分类
- 收入：工资收入、项目资金、奖学金、兼职收入等
