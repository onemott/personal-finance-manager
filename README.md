# SCCIPC智能车实验室订单与账单管理系统

一个功能完整的SCCIPC智能车实验室财务管理系统，支持收支记录、订单管理、财务分析等功能。

## 功能特性

- 💰 财务概览仪表板
- 📊 多维度记录管理（收入/支出/订单）
- 🔍 强大的筛选和搜索功能
- 📈 财务分析和趋势统计
- 💾 本地数据存储和备份
- 📱 响应式设计，支持移动设备

## 在线访问

访问地址：`https://你的用户名.github.io/仓库名`

## 本地运行

1. 克隆项目到本地
2. 在项目目录下运行：`python -m http.server 8080`
3. 打开浏览器访问：`http://localhost:8080`

## 部署说明

### GitHub Pages 部署步骤：

1. 创建 GitHub 仓库
2. 上传所有文件（index.html, style.css, script.js）
3. 在仓库设置中启用 GitHub Pages
4. 选择 main 分支作为源
5. 几分钟后即可通过 GitHub Pages 链接访问

### 其他部署选项：

- **Netlify**: 拖拽文件夹即可部署
- **Vercel**: 连接 GitHub 仓库自动部署
- **Surge.sh**: 命令行快速部署
