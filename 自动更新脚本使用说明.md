# 🤖 自动更新脚本使用说明

## 📋 脚本功能
`自动更新脚本.bat` 是为SCCIPC智能车实验室财务系统设计的一键更新工具，可以自动将支付宝和微信账单导入功能部署到GitHub仓库。

---

## 🚀 使用方法

### 1. 准备工作
- ✅ 确保电脑已安装Git
- ✅ 确保GitHub账户有仓库推送权限
- ✅ 将所有更新文件放在同一目录下

### 2. 运行脚本
1. 📂 双击 `自动更新脚本.bat` 文件
2. 📝 **按回车键使用默认仓库**，或输入你的GitHub仓库URL
3. ✅ 确认更新文件列表
4. 🚀 等待自动完成更新

### 3. 默认仓库设置
- **默认URL**: `https://github.com/onemott/personal-finance-manager.git`
- **对应网站**: `https://onemott.github.io/personal-finance-manager`
- **使用方法**: 直接按回车键即可使用默认设置

## 🛠️ 故障排除

### 常见问题

**Q: Git环境检查失败**
```
❌ 未检测到Git，请先安装Git或使用GitHub Web界面更新
```
**解决方案**: 
- 下载安装Git: https://git-scm.com/
- 或使用GitHub Web界面手动更新

**Q: 仓库克隆失败**
```
❌ 克隆仓库失败，请检查URL和网络连接
```
**解决方案**:
- 检查网络连接
- 确认仓库URL正确
- 检查GitHub访问权限

**Q: 推送失败**
```
❌ 推送失败，可能需要身份验证
```
**解决方案**:
- 配置Git用户名和邮箱
- 设置GitHub访问令牌
- 使用GitHub Desktop
- 或使用GitHub Web界面

### 手动配置Git身份验证
```bash
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"
```
