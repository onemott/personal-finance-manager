# 支付宝和微信账单导入使用指南

## 🎯 功能概述

SCCIPC智能车实验室财务管理系统现已支持：
- ✅ **支付宝交易明细** (CSV格式)
- ✅ **微信支付账单** (XLSX格式)
- ✅ **系统JSON数据** (原有功能)

## 📱 支付宝账单导入

### 1. 获取支付宝账单
1. 打开支付宝APP
2. 进入 **我的** → **账单**
3. 点击右上角 **···** → **开具交易流水证明**
4. 选择时间范围，点击 **申请**
5. 等待邮件通知，下载CSV文件

### 2. 导入步骤
1. 在系统中点击 **数据管理**
2. 选择 **支付宝账单** 导入选项
3. 选择下载的CSV文件
4. 系统自动解析并导入交易记录

### 3. 支持的数据格式
- **文件格式**: CSV (UTF-8编码)
- **必需字段**: 交易时间、商品说明、交易金额
- **可选字段**: 交易状态、收/支类型
- **自动分类**: 根据商品描述智能分类

## 💬 微信账单导入

### 1. 获取微信账单
1. 打开微信APP
2. 进入 **我** → **服务** → **钱包**
3. 点击 **账单** → **常见问题** → **下载账单**
4. 选择 **用于个人对账** 
5. 选择时间范围，申请下载
6. 通过邮件接收XLSX文件

### 2. 导入步骤
1. 在系统中点击 **数据管理**
2. 选择 **微信账单** 导入选项
3. 选择下载的XLSX文件
4. 系统自动解析并导入交易记录

### 3. 支持的数据格式
- **文件格式**: XLSX/XLS
- **必需字段**: 交易时间、商品、金额
- **可选字段**: 当前状态、收/支类型
- **自动分类**: 根据商品描述智能分类

## 🔄 智能分类规则

### 收入分类
- **工资收入**: 包含"工资"、"薪资"关键词
- **奖金**: 包含"奖金"、"红包"关键词
- **其他收入**: 转账、收款等

### 支出分类
- **餐饮**: 餐厅、外卖、美团、饿了么等
- **交通**: 地铁、公交、打车、滴滴等
- **购物**: 商城、超市、淘宝、京东等
- **娱乐**: 电影、游戏、KTV、直播等
- **生活缴费**: 水电、话费、房租、物业等
- **医疗**: 医院、药店、挂号等
- **教育**: 学费、培训、图书等
- **其他支出**: 未匹配的支出

## ⚠️ 注意事项

### 数据安全
- ✅ 所有数据在本地处理，不上传到服务器
- ✅ 导入后原始文件可以删除
- ✅ 支持数据导出备份

### 文件要求
- **支付宝**: 必须是官方导出的CSV文件
- **微信**: 必须是官方导出的XLSX文件
- **编码**: 支持UTF-8和GBK编码
- **大小**: 建议单个文件不超过10MB

### 常见问题

**Q: 导入失败怎么办？**
A: 检查文件格式是否正确，确保是官方导出的账单文件

**Q: 分类不准确怎么办？**
A: 导入后可以手动编辑记录，修改分类和描述

**Q: 重复导入会怎样？**
A: 系统会添加所有记录，建议导入前清理重复数据

**Q: 支持哪些时间范围？**
A: 支持任意时间范围，建议按月或按季度导入

## 🎯 实验室使用建议

### 对于SCCIPC智能车实验室：

1. **项目经费管理**
   - 按项目分别导入相关支付记录
   - 使用备注字段标注项目名称

2. **设备采购记录**
   - 导入设备采购相关的支付记录
   - 手动调整分类为"设备采购"

3. **日常开支统计**
   - 定期导入实验室相关支出
   - 生成月度/季度财务报表

4. **多人协作**
   - 各成员分别导入个人支付记录
   - 统一汇总到实验室账户

1. 检查导入的记录数量
2. 验证金额和时间是否正确
3. 确认分类是否合理

### 数据整理
1. 编辑不准确的分类
2. 添加详细的备注信息
3. 删除无关的交易记录

### 数据分析
1. 查看财务概览统计
2. 按分类分析支出结构
3. 生成时间趋势报表
